# CS12 Course Portfolio

Welcome to your Computer Science 12 course portfolio! This repository will contain all your work for the semester.

## Repository Structure

- **notes/** - Links to class notes and resources
- **unit1-foundations/** - Variables, input/output, basic math
- **unit2-conditionals/** - If/else statements, logic
- **unit3-loops/** - While and for loops
- **unit4-functions/** - Methods and organization
- (More units added throughout the semester)

## Workflow

### At the start of each unit:
1. **Pull updates** from GitHub to get new unit materials

In VS Code: Click the down arrow

### During class:
1. Complete classwork in the `classwork/` folder
2. Work on assignments in the `assignments/` folder
3. Save often (Ctrl+S)

### At the end of class:
1. **Commit your work:**
 - Stage changes (+ button)
 - Write commit message
 - Commit (button)
2. **Push to GitHub:**
 - Click "Sync Changes" or the up arrow.

## Help

- **Forget how to commit?** See the Quick Reference Card
- **Can't push?** Ask me for help.
- **Lost your token?** - You'll need to generate a new one.

## Grading

I will have access to this repository and can see all your work. Make sure to:
- Commit regularly (shows you progress)
- Push at the end of each class (backs up your work)
- Write clear commit messages (helps you and me make sense of your work)
